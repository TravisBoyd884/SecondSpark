### Connect E-commerce Platforms with SecondSpark

*This folder contains API interfaces for e-commerce websites such as Ebay and Etsy.*

To connect to the Ebay api, you must supply developer account credentials
in a file named `.env` in this folder. In this file you must place the
client_secret and client_id.

**SYNTAX:**

`EBAY_CLIENT_ID=[your-client-id]`

`EBAY_CLIENT_SECRET=[your-client-secret]`

`EBAY_ENV=[environment-type]`

`ETSY_CLIENT_ID=[your-client-id]`

`ETSY_CLIENT_SECRET=[your-client-secret]`

`ETSY_ENV=[environment-type]`

`ETST_ACCESS_TOKEN=[token]`

`ETSY_SHOP_ID=[id]`

*Evironment types (for ETSY_ENV or EBAY_ENV) may be sandbox or production*

*Etsy access token and shop id are optional*
